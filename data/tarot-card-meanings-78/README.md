  # Complete Tarot Card Meanings Dataset

  A comprehensive dataset of all 78 tarot cards with upright, reversed, love, and career interpretations.

  ## Dataset Description

  This dataset contains detailed meanings for all 78 tarot cards including the 22 Major Arcana and 56 Minor Arcana cards across four suits (Wands, Cups, Swords, Pentacles).

  ### Source
  Card meanings compiled from [Deckaura Tarot Guide](https://deckaura.com/blogs/guide/tarot-card-meanings) — a comprehensive tarot education resource.

  ## Features

  | Column | Description |
  |--------|-------------|
  | card_number | Card position (0-77) |
  | card_name | Full card name |
  | arcana | Major or Minor Arcana |
  | suit | Wands, Cups, Swords, or Pentacles |
  | element | Fire, Water, Air, or Earth |
  | upright_meaning | Keywords for upright position |
  | reversed_meaning | Keywords for reversed position |
  | love_meaning | Love and relationship interpretation |
  | career_meaning | Career and work interpretation |
  | yes_or_no | Yes, No, or Maybe |
  | zodiac_sign | Associated zodiac or planet |
  | guide_url | Link to full card guide |
